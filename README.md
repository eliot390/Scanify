# Scanify - Barcode Scanner App
### Objective:
The main objective of this project is to develop an Android based app which can scan barcodes to fetch product information and purchasing options:
-	Allow users to create accounts for personalization
- Save products to personal database for future purchase
- Compare prices of online & retail locations
---
### Methods: 
The following methods will be used to achieve the project objectives:
-	Auth0 authentication
- Barcode API
- SQLite database
